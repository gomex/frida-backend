# NEWS SERVICE 2.0 [![Build Status](https://snap-ci.com/marighella/news-service-2.0/branch/master/build_image)](https://snap-ci.com/marighella/news-service-2.0/branch/master)

## COMMANDS

Run pm2

```
npm run pm2:startup
```

Show pm2 logs

```
npm run pm2:logs
```

Monitoring pm2 apps

```
npm run pm2:monit
```

Describe pm2 apps

```
npm run pm2:describe
```
