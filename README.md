## COMMANDS

Run pm2

> In this app `npm start` command uses this command in background

```
npm run pm2:startup
```

Show pm2 logs

```
npm run pm2:logs
```

Monitoring pm2 apps

```
npm run pm2:monit
```

Describe pm2 apps

```
npm run pm2:describe
```
